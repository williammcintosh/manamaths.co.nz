{\LARGE \textbf{L02 Equivalent fractions}}\\[0.35em]
Find or identify fractions that are equivalent.

\vspace{2.6em}
{\LARGE \textbf{Foundation}}
\begin{multicols}{3}

  itemsep=7.8em,
  label=\textbf{\arabic*.},

  labelsep=0.75em,
  leftmargin=!,
  align=left
]
\item Write a fraction equivalent to $\frac{1}{2}$.
\item Write a fraction equivalent to $\frac{2}{3}$.
\item Write a fraction equivalent to $\frac{3}{4}$.
\item Write a fraction equivalent to $\frac{4}{5}$.
\item Are $\frac{1}{2}$ and $\frac{2}{4}$ equivalent?
\item Are $\frac{3}{5}$ and $\frac{6}{10}$ equivalent?
